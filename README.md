# -كود سورس لبرنامج صغير
يوقوم بتحويل من النصوص إلى النظام السداسي عشر
و إظافة دالة التحويل من السداسي عشر إلى نصوص
قد تحتاج هذا البرنامج في حالة كتابة لنصوص في برنامجك بالغة العربية
و أرت تشفير الكود سوس الخاص بالأتويت او لغة أخرى
حيث انه بعد التشفير تصبح اللغة غير مقرئة
هذا البرنامج يسهل عليك نسخ فقط النص العربي و غعادة لصقه فقط
حيث انه عند النسخ يحول من النص إلى النظام السادس عشر
و وضغها في الحافظة
و ند اللصق تصق معها دالة التحويل أي عند تنفيذ البرنامج يتم الترجمة من السادس عشر إلى النص
وهذا لتفادي مشل اللغة العربية بعد تشفير الكود سورس
